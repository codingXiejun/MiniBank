# MiniBank

* 本项目是一个基于SpringBoot+Mybatis+Bootstrap+Redis架构的网银账户系统，模拟银行账户的日常操作，比如银行卡管理，实名认证管理，转账以及日志记录等功能
* 该项目使用SpringBoot微服务，免去了繁琐的配置过程；使用了`热部署`，提高了开发效率；通过include指令将页面分割，逻辑分离，提高了代码的简洁性；
* 遇到的问题以及解决办法：
	* 提高账户的安全性：使用`MD5`加密算法
	* 如何利用`CAS`算法保证转账操作的原子性(设计中)
	* 使用`Redis`对非帐号信息进行缓存：通过`Jedis`的高度封装类`RedisTemplate`类对没有缓存过的非账号信息进行缓存
